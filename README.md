# 学习通超级工具

一个用于超星学习通网页端的 Tampermonkey 用户脚本，集 **题目采集 / 导出 / AI 答题 / 回填** 与 **自动刷课（视频自动播放、保活、自动下一节、章节测验 AI 作答）** 于一体，统一浮窗面板控制。

脚本由两大模块组成：

- **题目采集 / 导出 / AI 答题模块**：采集题目、选项与正确答案，导出为 Excel、Markdown 或 JSON；可选调用 DeepSeek 等 OpenAI 兼容 API 自动生成答案并回填；
- **自动刷课模块**：在课程学习页自动接管视频播放，倍速观看、卡顿自动恢复，播完自动进入下一节；遇到章节测验可调用 AI 自动作答并提交。

> 默认只采集题目、选项和正确答案；AI 自动答题与自动刷课为可选功能，需自行配置 API Key 并按需开启。

## 功能

### 题目采集与导出

- **逐题换页采集**：保存当前题后自动进入下一题，页面刷新后继续任务；
- **长卷同页采集**：一次解析当前页面已经加载的全部题目；
- **随堂练习采集**：直接读取随堂练习页面已经加载的 `questionList` 数据；
- **AI 自动答题**：调用 DeepSeek 等 OpenAI 兼容 API 生成答案并自动回填（支持自动翻页）；
- **已采集答案回填**：将之前采集到的正确答案自动填入当前页面对应题目；
- **自动显示答案**：未作答题存在"显示答案"时，尝试进入答案页后继续采集；
- **断点恢复**：刷新、关闭页面或手动暂停后，可继续未完成任务；
- **大题量支持**：适用于 500 题等大型题库；
- **进度隔离**：不同试卷或练习活动分别保存状态；
- **循环保护**：无法确认题号或重复跳转时自动停止，避免无限刷新；
- **多格式导出**：支持 Excel、Markdown 和 JSON。

### 自动刷课

- **自动播放视频**：进入课程学习页后自动接管并播放视频；
- **倍速观看**：可在面板中设置播放倍速（默认 1.5×）；
- **播放保活**：视频卡顿、暂停或异常中断时自动恢复播放，静音兜底；
- **自动下一节**：当前视频播完后自动切换到下一节继续学习；
- **章节测验 AI 作答**：检测到章节测验时调用 AI 自动填写答案；
- **自动提交测验**：AI 作答完成后自动提交（可在面板关闭）；
- **找不到视频时跳过**：非视频步骤可跳过，或优先尝试 AI 做测验；
- **统一面板**：刷课控制已合并到主浮窗，与采集 / 答题共用同一面板。

## 支持的页面模式

### 1. 逐题换页模式

适用于一次只显示一道题、点击"下一题"后页面重新加载的自测题或试卷回顾页面。

脚本会依次：

1. 读取当前题目与正确答案；
2. 将结果和进度保存到浏览器本地；
3. 自动进入下一题；
4. 页面重新加载后恢复任务；
5. 从下一道缺失题继续采集。

### 2. 长卷同页模式

适用于多道题全部显示在同一页面中的试卷。

脚本会自动滚动页面等待懒加载内容出现，再逐题解析当前页面，不点击"下一题"。

如果某道题的"显示答案"会离开整张长卷页面，该题会被跳过，避免破坏整页采集流程。

### 3. 随堂练习模式

适用于超星随堂练习的答题结果页面，例如：

```text
https://mobilelearn.chaoxing.com/page/quiz/stu/quizStudentQuestion...
```

这类页面的题目由 Vue 组件渲染，页面文本可能显示为繁体"正確答案"，普通 DOM 选择器不一定能稳定读取。

脚本会直接读取当前页面已经加载的题目数据，包括：

- `content`：题干；
- `options`：选项；
- `rightAnswer`：正确答案；
- 活动名称：作为导出文件标题。

随堂练习模式不会点击页面按钮，也不需要逐题跳转。打开答题结果页面后，点击"开始采集"即可一次性保存当前活动的全部题目。

### 4. 课程学习页（自动刷课）

适用于课程学习页：

```text
https://mooc2-ans.chaoxing.com/mycourse/studentstudy
```

脚本会在该页面自动接管视频播放、倍速、保活、自动下一节，并处理章节测验。

注意：`mooc2-ans.chaoxing.com` 外壳页经常只是一个 iframe 外壳，真正的题目或视频在 `mobilelearn.chaoxing.com` 的 frame 内。脚本会检测外壳页并退出，交给 iframe 内匹配到的脚本实例运行，避免重复挂载面板或只能扫到底部确认框。

## AI 自动答题

脚本可以通过 DeepSeek 等 OpenAI 兼容 API 自动为当前页面的题目生成答案并回填，也用于章节测验的自动作答。

### 配置

1. 点击右下角"答"按钮打开面板；
2. 在"设置"区域的 AI 配置中填写：
   - **API Key**：你的 DeepSeek 或其他 OpenAI 兼容服务的密钥；
   - **API 地址**：默认 `https://api.deepseek.com`（官方 DeepSeek API 会自动去除 `/v1` 后缀，并映射旧模型名）；
   - **模型**：默认 `deepseek-v4-flash`；
   - **填充间隔**：每题之间的延迟，默认 500ms；
   - **DeepSeek V4 打开思考**：可选开启更慢但更深入的推理模式（更耗额度）；
3. 可点击"测试连接"验证配置；
4. API Key 使用 `GM_setValue` 安全存储，不会写入 `localStorage`。

### 使用

- **AI 自动答题（自动翻页）**：对当前页面已识别的题目调用 API 生成答案并自动填写，逐题换页试卷会按题号自动作答并保存进度；
- **回填已采集答案**：将之前采集到的正确答案回填到当前页面对应的题目中；
- **章节测验 AI 作答**：在自动刷课过程中检测到章节测验时自动调用 AI 填写并（可选）提交。

### 权限说明

AI 功能需要以下额外权限：

- `GM_xmlhttpRequest`：向 AI API 发送请求；
- `GM_getValue` / `GM_setValue` / `GM_deleteValue`：安全存储 API Key；
- `unsafeWindow`：访问页面上下文；
- `@connect api.deepseek.com` / `@connect *`：允许连接 AI API 服务。

## 自动刷课使用方法

1. 进入课程学习页（`/mycourse/studentstudy`）；
2. 点击右下角"答"按钮打开面板；
3. 在"自动刷课"区域按需开启：
   - **启用自动刷课**：总开关；
   - **自动播放视频**：自动接管视频播放；
   - **AI 做章节测验**：遇到章节测验时调用 AI 作答；
   - **自动提交测验**：AI 作答后自动提交；
   - **找不到视频时跳过**：关闭后会优先尝试 AI 做测验；
   - **倍速**：设置播放倍速；
4. 点击"立即启动"开始，"暂停"停止。

### 自动刷课参数

| 参数 | 默认值 | 说明 |
|---|---:|---|
| 启用自动刷课 | 开 | 总开关 |
| 自动播放视频 | 开 | 自动接管视频 |
| AI 做章节测验 | 开 | 章节测验用 AI 作答 |
| 自动提交测验 | 开 | AI 作答后自动提交 |
| 找不到视频时跳过 | 关 | 关闭则优先 AI 做测验 |
| 倍速 | 1.5 | 播放倍速 |
| 重试间隔 | 2000ms | 播放失败重试间隔 |
| 最大重试 | 10 | 播放失败最大重试次数 |
| 视频检查间隔 | 1000ms | 保活检查频率 |
| 切换延迟 | 3000ms | 进入下一节前延迟 |

## 安装前：开启"允许用户脚本"

这是最容易遗漏的一步。

即使 Tampermonkey 已安装并显示"已启用"，浏览器扩展详情页中的 **"允许用户脚本"** 没有开启时，脚本仍可能完全不运行。

### Microsoft Edge

1. 在地址栏打开：

   ```text
   edge://extensions/
   ```

2. 找到 **Tampermonkey**；
3. 点击"详细信息"或"管理扩展"；
4. 打开 **"允许用户脚本"**；
5. 将网站访问权限设为"在所有网站上"，或至少允许访问 `chaoxing.com`；
6. 返回超星页面，按 `Ctrl + F5` 强制刷新。

### Google Chrome

1. 在地址栏打开：

   ```text
   chrome://extensions/
   ```

2. 找到 **Tampermonkey**；
3. 点击"详情"；
4. 打开 **"允许用户脚本"**；
5. 确认 Tampermonkey 可以访问超星网站；
6. 返回超星页面，按 `Ctrl + F5` 强制刷新。

> "脚本已启用"和"允许用户脚本"是两个不同的开关，两个都需要开启。

## 安装

### 一键安装

[点击安装学习通超级工具](https://raw.githubusercontent.com/zenghaolinz/chaoxing-learning-super-tool/main/chaoxing-learning-super-tool.user.js)

Tampermonkey 应自动打开安装页面。点击"安装"后刷新超星页面即可。

### 手动安装

1. 打开 Tampermonkey 管理面板；
2. 点击"添加新脚本"；
3. 删除编辑器中的默认内容；
4. 将 `chaoxing-learning-super-tool.user.js` 的完整内容粘贴进去；
5. 按 `Ctrl + S` 保存；
6. 刷新超星题目页面。

## 使用方法

进入支持的超星页面后，右下角会出现蓝色"答"按钮。

### 开始采集

1. 点击右下角"答"按钮；
2. 点击"开始采集"；
3. 脚本自动识别页面模式；
4. 等待采集完成；
5. 选择 Excel、Markdown 或 JSON 导出。

逐题采集期间，请不要手动点击题号、"上一题"或"下一题"。

### 扫描当前页

"扫描当前页"只处理当前页面已加载的数据：

- 普通试卷页：读取当前 DOM 中的题目；
- 随堂练习页：读取 Vue `questionList` 中的全部题目。

### 暂停与恢复

- 点击"暂停并保存"，或按 `Esc`，可以停止自动跳转；
- 已采集结果保存在当前浏览器的 `localStorage` 中；
- 重新打开同一份试卷后，点击"继续采集"即可恢复；
- 随堂练习模式一次性完成，通常不需要断点恢复。

## 数据与隐私

- 题目数据只从当前超星页面已加载的内容中读取；
- 随堂练习模式读取页面现有的 Vue 数据，不额外请求接口；
- 采集进度保存在浏览器本地；
- 导出文件在浏览器本地生成；
- 脚本不需要账号密码或 Cookie；
- AI 功能需要自行提供 API Key，Key 通过 `GM_setValue` 存储在本地，不会上传或外传；
- AI 答题请求仅发送到你配置的 API 地址；
- Excel 导出使用 SheetJS 0.18.5，Markdown 和 JSON 不依赖外部组件。

## 常见问题

### 页面没有出现蓝色按钮

依次检查：

1. Tampermonkey 是否启用；
2. 是否打开"允许用户脚本"；
3. 是否允许 Tampermonkey 访问 `chaoxing.com`；
4. 当前脚本是否启用；
5. 页面是否已经加载出题目；
6. 是否按 `Ctrl + F5` 刷新；
7. 是否同时启用了其他同类脚本。

### 随堂练习页面没有识别

1. 确认打开的是"答题结果"页面，而不是活动列表；
2. 等待题目完整显示后再点击"开始采集"；
3. 确认 URL 属于 `mobilelearn.chaoxing.com`；
4. 按 `Ctrl + F5` 刷新；
5. 如果页面没有向当前账号提供 `rightAnswer`，脚本无法导出正确答案。

### AI 答题不工作

1. 确认已在设置中填写 API Key；
2. 确认 API 地址正确（默认 `https://api.deepseek.com`）；
3. 确认 API Key 有余额且未过期；
4. 确认模型名称正确；
5. 检查 Tampermonkey 是否允许跨域请求；
6. 如果提示 429 错误，调大填充间隔后重试。

### 自动刷课不生效

1. 确认进入的是课程学习页（`/mycourse/studentstudy`），脚本仅在该页面接管视频；
2. 确认面板中"启用自动刷课"和"自动播放视频"已开启；
3. 点击"立即启动"手动触发；
4. 确认页面存在 `video` 元素或视频 iframe；
5. 若当前步骤是章节测验而非视频，确认"AI 做章节测验"已开启并配置了 API Key；
6. 按 `Ctrl + F5` 刷新后重试。

### 一直刷新同一道题

1. 按 `Esc` 暂停；
2. 禁用其他旧版或同类脚本；
3. 刷新当前页面；
4. 再点击"继续采集"。

### 部分题目没有正确答案

可能原因包括：

- 当前账号没有查看答案的权限；
- 页面没有"显示答案"按钮；
- 答案由图片、公式或特殊组件渲染；
- 超星更新了页面结构；
- 长卷页面中的"显示答案"会离开当前试卷；
- 随堂练习的题目数据中没有 `rightAnswer`。

无法读取的题目会记录为跳过，不会阻塞整套采集。

### Excel 提示组件未加载

Excel 导出依赖 cdnjs 上的 SheetJS。请检查网络、广告拦截器或其他扩展是否阻止资源加载。

Markdown 和 JSON 可以作为备用。

## 项目结构

```text
chaoxing-learning-super-tool/
├── chaoxing-learning-super-tool.user.js
├── README.md
├── CHANGELOG.md
├── LICENSE
└── .gitignore
```

脚本内部主要模块：

**题目采集 / 导出 / AI 答题模块**

- `ChaoxingAdapter`：识别 DOM 页面和 Vue 随堂练习数据；
- `SessionStore`：按试卷或活动保存进度；
- `Collector`：处理采集模式；
- `Exporter`：生成 Excel、Markdown 和 JSON；
- `AIClient`：调用 AI API 生成答案；
- `AutoFiller`：匹配题目并自动回填答案；
- `AppUI`：使用 Shadow DOM 构建操作面板。

**自动刷课模块**

- `StudyAutoPlayer`：接管视频播放、倍速、保活、自动下一节；
- 章节测验检测与 AI 作答（`_answerQuizIfPresent` 等）；
- 跨 iframe 上下文遍历（`_walkFrameContexts`）。

## 开发

语法检查：

```bash
node --check chaoxing-learning-super-tool.user.js
```

发布新版本时，需要同步修改脚本内三处版本号：

```javascript
// @version      1.5.1
```

```javascript
version: '1.5.1',           // APP.version
```

```javascript
version: '1.5.1',           // StudyAutoPlayer MODULE.version
```

推送到 `main` 分支后，Tampermonkey 会通过脚本中的 `@updateURL` 检查更新。

## 已知限制

- 超星不同课程和页面可能使用不同结构；
- 平台更新后，部分选择器或 Vue 字段可能需要重新适配；
- 长卷页面中会发生跳转的"显示答案"无法安全批量处理；
- 图片、公式和特殊题型可能无法完整导出；
- 随堂练习模式目前适配 Vue 2 页面中的 `questionList`、`content`、`options` 和 `rightAnswer`；
- AI 生成的答案可能不准确，请自行核实；
- 自动刷课仅在学习页接管视频，倍速受浏览器和平台限制；
- 脚本只能读取当前账号有权限查看的内容。
