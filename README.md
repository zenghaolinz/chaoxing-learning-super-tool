# 超星答案导出器

一个用于超星学习通网页端的 Tampermonkey 用户脚本，可采集试卷题目、选项与正确答案，并导出为 Excel、Markdown 或 JSON。

脚本同时兼容逐题换页和长卷同页两类页面，支持未作答题目的“显示答案”跳转、中断恢复及大题量题库采集。

> 默认只保存题目、选项和正确答案，不导出“我的答案”、答案解析、难易度或知识点。

## 功能

- 逐题换页采集：页面每次只显示一道题时，自动保存并进入下一题
- 长卷同页采集：整张试卷显示在一个页面时，一次解析全部已加载题目
- 自动显示答案：未作答题存在“显示答案”时，自动进入答案页面后继续采集
- 断点恢复：刷新、关闭页面或手动暂停后，可继续未完成的采集任务
- 大题量支持：适用于 500 题等大型题库
- 进度隔离：不同试卷分别保存采集状态
- 循环保护：无法确认题号或重复跳转时自动暂停，避免无限刷新
- 多格式导出：支持 Excel、Markdown 和 JSON
- 低权限运行：不读取账号密码，不申请跨域请求权限

## 支持的页面模式

### 逐题换页模式

适用于一次只显示一道题，点击“下一题”后页面重新加载的自测题或试卷回顾页面。

脚本会依次执行：

1. 读取当前题目与正确答案；
2. 将结果和采集进度保存到浏览器本地；
3. 自动进入下一题；
4. 页面重新加载后恢复任务；
5. 从下一道缺失题继续采集。

### 长卷同页模式

适用于多道题全部显示在同一页面中的试卷。

脚本会自动滚动页面，等待懒加载内容出现，然后逐题解析当前页面，不点击“下一题”。

如果某道未作答题的“显示答案”会离开整张长卷页面，该题会被跳过，避免破坏整页采集流程。

## 安装前：开启“允许用户脚本”

这是最容易遗漏的一步。

即使 Tampermonkey 已安装并显示“已启用”，浏览器扩展详情页中的 **“允许用户脚本”** 没有开启时，用户脚本仍可能完全不运行。

### Microsoft Edge

1. 在地址栏打开：

   ```text
   edge://extensions/
   ```

2. 找到 **Tampermonkey**；
3. 点击“详细信息”或“管理扩展”；
4. 打开 **“允许用户脚本”**；
5. 将网站访问权限设为“在所有网站上”，或至少允许访问 `chaoxing.com`；
6. 返回超星页面，按 `Ctrl + F5` 强制刷新。

### Google Chrome

1. 在地址栏打开：

   ```text
   chrome://extensions/
   ```

2. 找到 **Tampermonkey**；
3. 点击“详情”；
4. 打开 **“允许用户脚本”**；
5. 确认 Tampermonkey 可以访问超星网站；
6. 返回超星页面，按 `Ctrl + F5` 强制刷新。

> “脚本已启用”和“允许用户脚本”是两个不同的开关，两个都需要开启。

## 安装

### 一键安装

[点击安装超星答案导出器](https://raw.githubusercontent.com/zenghaolinz/chaoxing-answer-exporter/main/chaoxing-answer-exporter.user.js)

Tampermonkey 应自动打开安装页面。点击“安装”后刷新超星页面即可。

### 手动安装

1. 打开 Tampermonkey 管理面板；
2. 点击“添加新脚本”；
3. 删除编辑器中的默认内容；
4. 将 `chaoxing-answer-exporter.user.js` 的完整内容粘贴进去；
5. 按 `Ctrl + S` 保存；
6. 刷新超星试卷页面。

## 使用方法

进入包含题目的超星页面后，右下角会出现蓝色“答”按钮。

### 开始采集

1. 点击右下角“答”按钮；
2. 点击“开始采集”；
3. 脚本会自动识别逐题换页或长卷同页模式；
4. 等待采集完成；
5. 选择 Excel、Markdown 或 JSON 导出。

逐题采集期间，请不要手动点击题号、“上一题”或“下一题”。

### 扫描当前页

“扫描当前页”只读取当前页面已经显示的内容，不自动翻页，也不点击“显示答案”。

适合用于：

- 测试当前页面能否被识别；
- 只保存当前页面；
- 出现异常时检查题目选择器。

### 暂停与恢复

- 点击“暂停并保存”，或按 `Esc`，可以停止自动跳转；
- 已采集结果保存在当前浏览器的 `localStorage` 中；
- 重新打开同一份试卷后，点击“继续采集”即可从缺失题目恢复；
- 如果连续跳转仍无法确认当前题号，脚本会自动暂停，避免无限刷新。

## 数据与隐私

- 题目数据只从当前超星页面读取；
- 采集进度保存在浏览器本地；
- 导出文件在浏览器本地生成；
- 脚本不需要账号密码、Cookie 或 AI API Key；
- 脚本未申请 `GM_xmlhttpRequest` 等跨域请求权限；
- Excel 导出使用 SheetJS 0.18.5，Markdown 和 JSON 导出不依赖外部组件。

## 常见问题

### 页面没有出现蓝色按钮

依次检查：

1. Tampermonkey 是否启用；
2. 是否打开“允许用户脚本”；
3. 是否允许 Tampermonkey 访问 `chaoxing.com`；
4. 当前脚本是否处于启用状态；
5. 页面是否已经加载出题目；
6. 是否按 `Ctrl + F5` 刷新；
7. 是否同时启用了其他同类脚本。

### 一直刷新同一道题

1. 按 `Esc` 暂停；
2. 禁用其他旧版或同类脚本；
3. 刷新当前页面；
4. 再点击“继续采集”。

脚本内置跳转循环保护，连续失败后会自动暂停。

### 部分题目没有正确答案

可能原因包括：

- 当前账号没有查看答案的权限；
- 页面没有“显示答案”按钮；
- 答案由图片、公式或特殊组件渲染；
- 超星更新了页面结构；
- 长卷页面中的“显示答案”会离开当前试卷。

无法读取的题目会记录为跳过，不会阻塞整套采集。

### Excel 提示组件未加载

Excel 导出依赖 cdnjs 上的 SheetJS。请检查网络、广告拦截器或其他扩展是否阻止资源加载。

Markdown 和 JSON 导出不依赖 SheetJS，可以作为备用。

## 项目结构

```text
chaoxing-answer-exporter/
├── chaoxing-answer-exporter.user.js  # 用户脚本
├── README.md                          # 使用说明
├── CHANGELOG.md                       # 版本记录
├── PUBLISH.md                         # 发布流程
├── LICENSE                            # 开源许可证
└── .gitignore
```

脚本内部主要模块：

- `ChaoxingAdapter`：识别页面结构并提取题目内容
- `SessionStore`：按试卷保存采集进度
- `Collector`：处理逐题换页和长卷同页采集
- `Exporter`：生成 Excel、Markdown 和 JSON
- `AppUI`：使用 Shadow DOM 构建操作面板

## 开发

语法检查：

```bash
node --check chaoxing-answer-exporter.user.js
```

发布新版本时，需要同步修改用户脚本中的两处版本号：

```javascript
// @version      1.0.0
```

```javascript
version: '1.0.0'
```

推送到 `main` 分支后，Tampermonkey 会通过脚本中的 `@updateURL` 检查更新。

## 已知限制

- 超星不同课程和页面可能使用不同 DOM 结构；
- 平台更新后，部分选择器可能需要重新适配；
- 长卷页面中会发生跳转的“显示答案”无法安全批量处理；
- 图片、公式和特殊题型可能无法完整导出；
- 脚本只能读取当前账号有权限查看的内容。
